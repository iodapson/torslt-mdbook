### Closure Traits

| Trait(s) | Category | Operator(s) | Description                  |
| -------- | -------- | ----------- | ---------------------------- |
| `Fn`     | closure  | `(...args)` | immutable closure invocation |
| `FnMut`  | closure  | `(...args)` | mutable closure invocation   |
| `FnOnce` | closure  | `(...args)` | one-time closure invocation  |
